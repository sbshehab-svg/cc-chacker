# Stripe Donation Gateway (CC Checker)

এই প্রোজেক্টটি Stripe API ব্যবহার করে একটি Payment Method তৈরি করে এবং Palestine Charity সাইটে ডোনেশন সম্পন্ন করার জন্য ডিজাইন করা হয়েছে।

## সেটআপ (Setup)

১. প্রথমে আপনার পিসিতে Python ইনস্টল করা থাকতে হবে।
২. প্রয়োজনীয় লাইব্রেরি ইনস্টল করতে নিচের কমান্ডটি রান করুন:

```bash
pip install -r requirements.txt
```
```render
gunicorn main:app
```
## ব্যবহার (Usage)

`main.py` ফাইলে আপনার কার্ডের ডিটেইলস এবং ইউজারের তথ্য আপডেট করুন এবং স্ক্রিপ্টটি রান করুন:

```bash
python main.py
```

## প্রোজেক্ট স্ট্রাকচার

- `main.py`: প্রধান লজিক এবং গেটওয়ে প্রসেস।
- `gateway.log`: প্রতিবার রান করার সময় কী হচ্ছে তার বিস্তারিত লগ এখানে জমা হবে।
- `requirements.txt`: প্রোজেক্টের ডিপেন্ডেন্সি লিস্ট।

## সাবধানতা

- এই কোডটি শুধুমাত্র শিক্ষামূলক উদ্দেশ্যে ব্যবহার করুন।
- সরাসরি লাইভ কী ব্যবহার করার সময় সাবধান থাকুন।
